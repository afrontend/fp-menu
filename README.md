## Installation

```sh
git clone https://github.com/afrontend/fp-menu.git
cd fp-menu
npm install
npm run release
```

## Usage

```js
# Linux
./dist/Fp-Menu\ 0.0.1.AppImage
./dist/Fp-Menu\ 0.0.1.AppImage --show-config-file > ~/.xmenu.json
```
